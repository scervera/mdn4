There are following settings that you can easily change (You can do advance customization but for that you need to aware from HTML/CSS):

  1. Layout

  1. Layout:- There are basically 2 type of layouts that you can control:
    a) Boxed
    b) Full Width

    a) Boxed: Boxed layout means there must be some fixed width for larger screen. If you want to enable Boxed layout then you need to add a class "boxed" in main div having id="wrapper". It will looks like: 

     <body>
	<div id="wrapper" class="boxed"> <!-- added class boxed -->
		<!-- Own HTML code to display information in visible area -->
	</div>
 </body>
    If you want to change width for boxed layout then you can edit responsive.css file & search for #wrapper.boxed. You can find this css at various places used because of "Media Query" & write width whatever you want. 
    	
 #wrapper.boxed {
		width: 768px;
		margin: 0 auto;
	}

    b) Full Width: Full width Layout means template will display at in full screen area.You will open this template at any resolution it will also display in 100%. To apply this layout you don't need to add any class with div having id "wrapper". It will looks like: 

     <body>
	<div id="wrapper"> 
	<!-- Own HTML code to display information in visible area -->
	</div>
 </body>

  2. Color:-Global colors are assigned in a file named "skin.less" placed under "assets/css" directory. You need to open this file in any HTML editor & look at below code: 
    @skinColor:#e7bd15;
    Changing color code from here will affect the global color on whole website.


  3. Font:- Global colors are assigned in a file named "skin.less" placed under "assets/css" directory. You need to open this file in any HTML editor & look at below code: 
    @fontFamily: 'Lato',sans-serif;
    Changing font family from here will affect the global color on whole website.

  4. Sticky Header:-There are 2 type of sticky header can be controlled in this template.
    a). Normal Header Navigation will always display static. It will not be visible on scroll up or down.
    b). Intelligent Sticky Header Navigation will display at top when you will scroll up but disappears when you scroll down.

    If you want to enable intelligent sticky header then you need to edit site.js file & search "Sticky header yes or no to activate function". The code looks like: 

     // Normal Header
 $('#header').attr('data-sticky', 'no')
 
 // Intelligent Sticky Header
 
 $('#header').attr('data-sticky', 'yes')
